# GitHub PR Template Configuration

This directory contains Terraform configurations for managing GitHub PR template settings in StepSecurity.

## Directory Structure

- `main/` - Main Terraform configuration for PR template management

## What is a PR Template?

The GitHub PR template resource allows you to customize the pull requests that StepSecurity automatically creates when security issues are detected in your repositories. You can configure:

- PR title and description
- Commit messages
- Labels
- Dynamic content using placeholders

## Quick Start

Navigate to the `main/` directory and follow the README instructions to get started.

## Use Cases

1. **Standardized PR Format**: Ensure all security PRs follow your organization's standards
2. **Custom Branding**: Add your organization's security documentation links
3. **Automated Labeling**: Apply consistent labels for security PRs
4. **Compliance Tracking**: Include compliance-related information in PR descriptions
